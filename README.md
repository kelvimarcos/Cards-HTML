
Página de apresentação de planos de hospedagem fictícia, criada com HTML, CSS e JavaScript puro. <br>
O projeto foi desenvolvido para treinar a criação de layouts responsivos, cards de planos e interações de botão com hover.

---

<br>

## 🚀 Tecnologias utilizadas

HTML5 → Estrutura semântica <br>
CSS3 → Estilização, responsividade e efeitos visuais <br>
JavaScript → Interações simples na interface <br>

<br>
<br>

## 💻 Funcionalidades

✔ Layout responsivo para mobile, tablet e desktop <br>
✔ Menu de navegação com destaque de item ativo <br>
✔ Cards de planos com informações organizadas <br>
✔ Botões com efeito hover <br>
✔ Estrutura visual moderna com foco em clareza <br>

<br>
<br>

## 🎯 Objetivo do Projeto

Praticar HTML semântico <br>
Trabalhar com CSS moderno (flexbox e organização visual) <br>
Melhorar a construção de interfaces responsivas <br>
Aplicar conceitos básicos de UX/UI <br>
