
Página de apresentação de planos de hospedagem fictícia, criada com HTML, CSS e JavaScript puro.
O projeto foi desenvolvido para treinar a criação de layouts responsivos, cards de planos e interações de botão com hover.

---

<br>

## 🚀 Tecnologias utilizadas

HTML5 → Estrutura semântica
CSS3 → Estilização, responsividade e efeitos visuais
JavaScript → Interações simples na interface

<br>
<br>

## 💻 Funcionalidades

✔ Layout responsivo para mobile, tablet e desktop <br>
✔ Menu de navegação com destaque de item ativo <br>
✔ Cards de planos com informações organizadas <br>
✔ Botões com efeito hover <br>
✔ Estrutura visual moderna com foco em clareza <br>

<br>
<br>

## 🎯 Objetivo do Projeto

Praticar HTML semântico
Trabalhar com CSS moderno (flexbox e organização visual)
Melhorar a construção de interfaces responsivas
Aplicar conceitos básicos de UX/UI
